# ecircdb-frontend
Frontend for the ecircdb

## Setup on your local machine

1. Clone the Repository
```shell
$ git clone https://github.com/EnsemblGSOC/ecircdb-frontend.git ecircdb-frontend
$ cd ecircdb-frontend
```
2. Install dependencies
```shell
$ yarn install
```
3. Start server
```shell
$ yarn start
```
4. Visit `localhost:3000`
